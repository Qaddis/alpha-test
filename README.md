<h1 align="center">Тестовое задание</h1>

Стек:

- TypeScript
- Next.js
- Zustand

Используются данные со стороннего апи - [DummyJSON](http://dummyjson.com/)

## Установка зависимостей

Перед первым запуском, сразу после клонирования репозитория, нужно установить все необходимые зависимости:

```bash
npm install
# or
yarn install
# or
pnpm install
# or
bun install
```

## Запуск

Для запуска в dev-режиме нужно воспользоваться одной из следующих команд:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
